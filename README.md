## 功能介紹
   點選任一個職業按下按紐送出，則收到一句對應職業的幹話 !
## 啓動方式

- 將專案 clone 到本地端

https://github.com/Steavn-Chen/dryTalk_Generator.git

- 進入到專案資料夾
```
- 安裝 npm
```
  執行 npm install
```
- 啓動專案
```
  npm run dev
```
-  看到終端機以下字串，表示成功。
```
  
  The web is running on http://localhost:3000
